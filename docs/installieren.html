R course

installieren

Installiere zuerst R selbst, danach Rstudio. Führe je nach Betriebssystem die folgenden Schritte aus.

Windows

  • installiere R
  • installiere Rstudio
  • ändere die .RData settings für Reproduzierbarkeit

Mac

Entweder

  • installiere homebrew
  • im Mac Terminal, führe aus: brew install --cask r und brew install --cask rstudio
  • ändere die .RData settings für Reproduzierbarkeit

oder

  • installiere Xquartz
  • lade von cran.r-project.org/bin/macosx das ‘latest release’ herunter, z.B. R-4.1.2.pkg
  • öffne die Datei mit ‘Installer’ und befolge die Anweisungen
  • installiere Rstudio
  • ziehe die App in den ‘Applications’-Ordner
  • ändere die .RData settings für Reproduzierbarkeit

Linux

Installiere R auf Linux Ubuntu (unterstützte Versionen), sodass sudo apt upgrade auf die neueste R Version aktualisiert, falls sie veraltet ist (Quellen: Kris Eberwein, Dean Attali, Lisa Tagliaferri):

  • öffne ein Terminal (STRG+ALT+T) und füge folgende Zeilen nacheinander ein (STRG+SHIFT+V).
    Auf Linux Mint, ersetze $(lsb_release -cs) mit z.B. focal gemäß Liste.
    Auf Debian, finde den Link hier.
sudo echo "deb https://cloud.r-project.org/bin/linux/ubuntu $(lsb_release -cs)-cran40/" | sudo tee -a /etc/apt/sources.list

Prüfe /etc/apt/sources.list manuell auf den richtigen Link (wie oben angegeben). Am Ende muss ein / stehen.
Für den nächsten Schritt können andere keys erforderlich sein, z.B. 95C0FAF38DB3CCAD0C080A7BDC78B2DDEABC47B7

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys E298A3A825C0D65DFD57CBB651716619E084DAB9
sudo apt update
sudo apt install r-base r-base-dev
  • lade Rstudio herunter
  • installiere das, oder auf Ubuntu 20.04 (siehe LinuxConfig):
sudo apt -y install r-base gdebi-core
sudo gdebi rstudio-2021.09.1-372-amd64.deb
  • ändere die .RData settings für Reproduzierbarkeit

Manjaro Linux

  • schalte AUR frei (Pamac - Einstellungen - Drittanbieter)
  • suche nach rstudio-desktop und installiere es. R wird automatisch mit installiert. Das kann eine Stunde dauern.
    (Anleitung von Frank de Boer)
  • ändere die .RData settings für Reproduzierbarkeit
