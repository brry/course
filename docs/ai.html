AI

pro AI in coding

  • quickly solves a problem
  • works in unfamiliar coding areas, concepts, languages
  • suggests unknown functions and approaches

contra AI in coding

  • doesn’t always follow best practices
  • is hard to maintain and has security issues (see below)
  • needs experienced senior developers to catch issues
  • lowers debugging skills
  • reduces learning to think in a structured manner (see below)

solution

  • learn basic programming yourself
  • develop good coding habits from the beginning
  • make mistakes, debug, struggle, research, think, build problem solving skills
  • implement AI-assisted code critically
